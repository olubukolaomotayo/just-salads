# Just Salads

A simple, self-contained web app for a home salad business: build an order by size, get a shopping list (whole packages to buy), and see cost, profit, and tax set-aside. Runs entirely in the browser. No server, no accounts, no data leaves the device.

**This public version ships with a neutral sample only.** Real recipes, prices, and suppliers are kept private and loaded on-device via the app's Restore feature.

## Use it
Open `index.html` in any browser, or the hosted link, then on a phone use Share, Add to Home Screen.

## Data
Everything you enter is saved in your browser (localStorage) on that device. Use Download backup / Restore to move data between devices and to keep a safe copy.
